# i2c2

```package
pins=github:bsiever/microbit-pxt-powerUtils
```

This extension can be used to detect the source of power (battery vs. USB)

**Micro:bit v2 only!!!**

# Setting the pins

```sig
pins.setI2CPins(sda : DigitalPin,  scl : DigitalPin) : void
```

Set the Data and Clock pins.
